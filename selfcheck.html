Self-check quiz

1. What is the HTML syntax to add an external JavaScript file?

To add an external JavaScript file to an HTML document, you use the script element with the src attribute.

2. Can you run JavaScript in a file with a JS extension in your browser?

Yes, you can run JavaScript in a file with a .js extension in your web browser.

3. How do you write a multiple-line comment in JavaScript?

In JavaScript, you can write multiple-line comments using the /* and */ characters.

4. What is the best way to remove a line of code from running that you might want to keep as you debug?

To temporarily remove a line of code from running while debugging, you can comment out the line.
